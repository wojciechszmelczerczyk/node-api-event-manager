# Delete event

## Description

Delete event of currently authorized user.

<b>URL :</b> `/event/:id`

<b>Method:</b> `DELETE`

<b>URL parameter:</b> `id=[string]`.

<b>Authorized:</b> `YES`

## Success Response

Code: `204 No content`

Condition: If event with provided correctly id exists.

### Context example

No content.
